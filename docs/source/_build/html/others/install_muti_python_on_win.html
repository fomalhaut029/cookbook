5.1. 前言¶

因为python 发展到version 3.x后,并不向下兼容，而目前很多的项目还是运行在python2.x版本中。因此 在同一个环境下会需要两个版本的运行环境。本文接下里将结束两种方式实现window下，python多版本的共存 问题

5.2.2. 采用python 自带的launcher来切换 【推荐】¶

Note

Python launcher是用于Windows中的一个实用程序，可帮助我们定位和执行不同版本的Python解释器。它允许脚本或者命令行指示特定的Python版本的首选项 也就是说他能自动的选择你指定的解释器来执行py脚本。 Python launcher 是3.3版本新增功能适用于 Windows 的 Python 启动器是一个实用组件，可帮助您定位和执行不同的 Python 版本。 它允许脚本（或命令行）为特定的 Python 版本指示首选项，定位并执行该版本。
