多系统启动U盘制作
制作过系统启动盘的人都知道，一个U盘只要被制作成启动盘后，里面的文件将是杂乱无章的，抛开我们的强迫症不说，主要是这有失我们的逼格，不符合我们作为“会装系统”的大佬的形象啊！于是我们如果能把一个U盘分为几个部分，一部分用于装我们的临时文件，一部分用于制作Windows启动盘，甚至还可以把一部分用以制作Linux启动盘。这样一来，好方便用的同时还透露一丝牛逼，何乐而不为呢？ 本教程适用于制作文件盘、Ubuntu启动盘、windows10启动盘同存于一个U盘的问题，同时启动分区隐藏，不影响正常的文件存储。

U盘使用规划与实施分区
规划
对于U盘，最常见的用途是存文件、制作各种启动盘。于是我们可以设计一部分用以存储文件、一部分用于制作Windows启动盘、一部分用以制作Linux启动盘，考虑到实际需要，我们可以对我们的U盘（这里针对32G的U盘）做出如下规划：

名称	第一分区	第二分区	第三分区
内容(卷标)	文件	Window10	Ubuntu18
文件系统	NTFS	NTFS	FAT32
大小	12288M（12G）	9216M（9G）	剩余部分
用途	装一些文件	作为Windows系统启动盘	作为Ubuntu系统启动盘
说明：

Windows系统的所需的存储空间在5G左右，这里我考虑到把一些常用的软件放里面，所以设计了9G的分区用以制作Windows启动盘
Ubuntu大小为2G左右，我也考虑到放一些Linux常用的软件，所以就设计了6G左右的分区用以制作Linux启动盘。
这里的第三个分区，也就是需要制作Linux启动盘那个分区，文件系统必须设置为FAT32，
不然在后面的启动盘制作过程中会报错：“syslinux 不支持NTFS引导”
（这里我也是踩了坑后探索出来的，没想通的是在单独制作Ubunttu启动盘的时候用NTFS就不会报错啊，怎么到这里就不行了呢？有知道的可以留言解答下哈！）
大家可以根据自己的情况酌情设置
实施分区
一个32G的U盘，为空或者里面的数据不重要（制作后里面的数据会丢失）
打开软件磁盘管理器（win+r 输入：diskmgmt.msc），进入磁盘管理器。
找到U盘所在的位置，右击格式化。
再右击依次添加分区，设置大小、文件系统类型如上表所示，同时如果想要命名逼格更高点，就在卷名处改好。
三个分区制作成功，也就意味着你的电脑中貌似又多出了两个盘，那是U盘被切分的原因，所以你明白为什么可以制作三总状态的U盘了吧！

使用UltralSO(软碟通)制作启动盘
隐藏掉不需要制作该启动盘的另外两个分区（关键）
很多人在制作启动盘的过程中会发现软碟通会默认把启动盘制作在U盘的第一个分区上，每次后面再制作时由覆盖掉前面的，所以这一步相当关键，这决定了多启动盘的制作是否成功。下面的操作会帮助你解决这个问题：

打开cmd或powershell，按如下命令行操作；
diskpart      #运行diskpart工具

lis dis       #列出所有磁盘

sel dis 1      #选中u盘，视你的磁盘大小判断选择数字，1是我的u盘

lis par          #列出所有分区

sel par 1     #选中需要用来存储文件的分区，1是我的打算用来存储文件的分区

detail par    #查看该分区详细信息

set id=16  #分区类型改回隐藏

detail par  #查看该分区详细信息，可见分区已显示被隐藏

sel par 2  #选中用于制作windowsd启动盘的分区

set id=06  #取消分区隐藏

active  #即可将分区设置为活动分区

detail par  #查看该分区详细信息，可见分区已被改为活动

sel par 3  #选中用于制作Ubuntu启动盘的分区

set id=16  #分区类型改为隐藏

detail par  #查看该分区详细信息可见已被隐藏

可见总体的思路是想要用哪个分区制作启动盘就将U盘中另外的分区都隐藏掉，并且设置要制作启动盘为 活动 状态，改方法适用于将另一个需要制作启动盘的分区。


回到UltralSO继续制作启动盘
用UltraISO打开下载好的系统ISO文件，将其写入移动硬盘的新分区。“启动”–“写入硬盘镜像”，选择“硬盘驱动器”为移动硬盘，“便捷启动”–“便捷写入”。



注意：在选择开始写入时不需要重新格式化，而且记得将“便捷启动”改为“便捷写入”


重复以上过程即可制作好另外一个启动盘了哦！！！

大功告成，记得归位
两个启动盘都装好后，记得把那两个启动盘的分区设置为隐藏，以便文件盘文件的存放哦！

sel par 1    #选择分区1即我们的文件盘分区

set id=06    #取消分区隐藏

active       #设置为活动

sel par 2   #选择分区2即windows启动盘分区

set id=16    #隐藏分区

sel par 3     # 选择分区3即Linux启动盘分区

set id=16     #隐藏分区
