CPU 测试
++++++++++++++++++++++++++

.. note:: 

    在KVM虚拟化的CPU使用机制中，虚拟机的vCPU在虚拟机内部实现的各种调度对于宿主机的CPU是透明，
    每个vCPU对于物理CPU来说仅仅相当于一个进程，通过不同虚拟机不同优先级的情况来将CPU的核心分配给vCPU独占使用。
