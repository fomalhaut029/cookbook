快捷键
=========

======================  ======================
快捷键                       描述
======================  ======================
Ctrl + Shift + L           批量修改变量
Shift + Alt + F            格式化代码
Alt + Shift + 下键          复制下一行
Shift + Ctl + O            查看函数列表
Shift + F12                查看函数引用
======================  ======================
